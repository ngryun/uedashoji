# 요나고미나미고등학교 설악고등학교 2026 국제교류 — 3D 갤러리

우에다 쇼지 사진미술관(高松伸 설계) 콘셉트의 Three.js 1인칭 가상 미술관.
2026년 7월 12–15일 국제교류 사진 218점과 영상 16점을 2층, 날짜별 4개 전시실에 전시.
Day 1·2는 1층, Day 3·4는 2층에 배치했으며 로비 계단으로 두 층을 오갈 수 있습니다.

## 실행

```bash
node tools/serve.mjs
# → http://localhost:8735
```

(브라우저 보안 정책상 로컬 서버가 필요합니다. `file://`로 직접 열면 사진이 로드되지 않습니다.
같은 Wi-Fi의 휴대폰에서 보려면 `http://<맥의IP>:8735`로 접속하세요.)

## GitHub Pages 배포

`main` 브랜치에 푸시하면 `.github/workflows/deploy-pages.yml`이 공개용 파일만 모아 자동 배포합니다.
원본 `img/`, 변환 추적 정보 `assets/.source-map/`, 로컬 설정은 Git 및 배포 대상에서 제외됩니다.

## 조작

| 데스크톱 | 모바일 |
|---|---|
| W A S D · 이동 | 왼쪽 조이스틱 · 이동 |
| 마우스 · 시점 | 오른쪽 드래그 · 시점 |
| SHIFT · 달리기 | RUN 버튼 (토글) |
| SPACE · 점프 | JUMP 버튼 |
| 숫자 1 / 2 · 층 이동 | 1F / 2F 버튼 |
| 사진 클릭 · 크게 보기 | 사진 탭 · 크게 보기 |
| ESC · 마우스 해제 | |

시작 화면의 **2D 사진 목록**에서는 3D 조작 없이 날짜별로 모든 사진과 영상을 탐색할 수 있습니다.

## 구조

```
index.html        진입점 (시작 화면, HUD, 모바일 UI)
main.js           Three.js 앱 (건축·물리·컨트롤·지연로딩)
manifest.json     사진/영상 목록 (tools/convert.sh가 생성)
lib/three.module.js  Three.js r166 (로컬 사본, 오프라인 작동)
assets/photos/    확대 감상용 JPEG (최대 1600px)
assets/thumbs/    3D 전시·2D 목록용 JPEG (최대 640px)
assets/videos/    웹용 MP4 (H.264 720p)
assets/backdrop.jpg  로비 창밖 다이센 풍경
assets/.source-map/ 원본 변경 추적 정보
tools/convert.sh  원본(img/) → assets/ 변환 스크립트 (변경 감지·재실행 안전)
tools/serve.mjs   정적 서버 (Range 지원 — 동영상 재생용)
img/              원본 (변경하지 않음)
```

## 사진을 추가/변경하려면

`img/` 날짜 폴더에 원본을 넣고 다시 실행:

```bash
tools/convert.sh
```

원본 경로·크기·수정 시각이 바뀐 항목만 다시 변환합니다. 중간에 사진을 추가하거나 교체해도 번호가 바뀐 결과물을 안전하게 갱신하며, `manifest.json`은 모든 변환이 성공한 뒤 교체됩니다. 전시실 크기와 배치는 사진 수에 맞춰 자동 계산됩니다.

## 메모

- 동영상 스크린은 9m 이내 접근 시 자동 재생(음소거), 가까운 2개만 동시 재생
- 1층은 Day 1·2, 2층은 Day 3·4 전시이며 로비 계단으로 이동 가능
- 1F / 2F 버튼과 숫자키 1 / 2는 각 층 로비로 바로 이동하는 보조 기능
- 사진은 거리순으로 모바일 최대 36장, 데스크톱 최대 56장만 GPU에 유지
- 전시 화면은 640px 썸네일을 사용하고 확대 보기에서만 1600px 원본 로드
- URL에 `?touch=1`을 붙이면 데스크톱에서도 터치 UI 강제 표시 (테스트용)
